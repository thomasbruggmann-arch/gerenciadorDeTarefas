# 📋 Gerenciador de Tarefas (Kanban)

Um gerenciador de tarefas simples e intuitivo no estilo **Kanban**, desenvolvido com **HTML, CSS e JavaScript**. Ideal para organizar atividades do dia a dia de forma visual e prática.

---

## ✨ Funcionalidades

- 📝 Adicionar novas tarefas  
- ❌ Excluir tarefas  
- 📊 Organização em três colunas:
  - **A Fazer**
  - **Em Andamento**
  - **Concluído**

---

## 🖼️ Layout

O sistema segue o modelo clássico de Kanban, permitindo visualizar rapidamente o progresso das tarefas conforme elas avançam entre as etapas.

---

## 🚀 Tecnologias Utilizadas

- HTML5  
- CSS3  
- JavaScript (Vanilla)

---

## 💡 Objetivo

Este projeto foi desenvolvido com o objetivo de praticar conceitos fundamentais de desenvolvimento web, como:

- Manipulação do DOM  
- Organização de layout  
- Interatividade com JavaScript puro
